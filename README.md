# rubik-cube

a rubik's cube game

## deploy

nginx:

```
    server {
        listen       80;
        server_name  localhost;

        ...

        ...


        location /rubik/ { # 末尾必须要有/
            alias   <项目路径>/rubik-cube/; # 末尾必须要有/
            autoindex on;
        }
```
